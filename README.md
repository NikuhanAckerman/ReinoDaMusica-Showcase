# Componentes necessários | Needed software
  * 1 - Java 17 ou acima | or above
  * 2 - Instância no MongoDB Atlas | Instance in MongoDB Atlas

# Guia de instalação (PT-BR)
  * 1 - Clone o repositório para sua máquina localmente;
  * 2 - Use npm install no terminal para instalar node_modules;
  * 3 - Coloque os dados de conexão à instância MongoDB Atlas para uma instância sua, em "src\main\resources\application.properties"
  * 4 - Use npm start para iniciar o frontend;
  * 5 - Inicie o backend compilando o código em "src\main\java\trabalhojava\buscadordemusicasbrasileiras\BuscadordemusicasbrasileirasApplication.java"
  * 6 - Acesse o app pela URL "localhost:3000"

# Installation guide (EN)
  * 1 - Clone the repository to your local machine;
  * 2 - Use npm install in the terminal to install node_modules;
  * 3 - Insert your MongoDB Atlas instance connection string in "src\main\resources\application.properties";
  * 4 - Use npm start to start the frontend;
  * 5 - Start the backend by compiling the code through "src\main\java\trabalhojava\buscadordemusicasbrasileiras\BuscadordemusicasbrasileirasApplication.java"
  * 6 - Access the web app in the URL "localhost:3000"
